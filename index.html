Welcome!

There are two ways to fill out an application with us. The first way is to download the PDF file and fill it out in the program/app of your choice. The second way is to fill out the PDF online.

Note

Elements

"Elements" is the collective term for the items you can add and position on your application. This includes text, images, and signatures.

Accessing settings for elements

The settings relevant to each element can be accessed by clicking/tapping on the element to select that element. The available options are available below the element after it is selected.

Moving elements

An element can be moved by dragging the icon in the element's settings.

Removing elements

An element can be removed or deleted from your application by clicking/tapping in the element's settings.

Text elements

Adding text

To add text to your application, you can click/tap on one of the provided text boxes (signified by a blue box), or you can add a new text box by clicking/tapping in the top bar.

Set a text box's font size

To change a text box's font size, edit the number next to the icon in the text's settings.

Signature elements

Adding a signature

To add your signature, click/tap in the top bar.

Writing a signature

Upon clicking/tapping to add or edit a signature, you will be presented with a canvas and some controls on the bottom right corner of the screen. Editing the number next to will change the width of all lines. and will undo and redo the last line draw, respectively. will add the signature to your application, will clear the canvas, and will close the canvas without adding the signature to your application.

Scaling a signature

To scale a signature, edit the number next to in the signature's settings.

Change a signature's line width

To change your signature's line width, edit the number next to in the signature's settings. This should be done if your signature is scaled so that it is still easily visible.

Editing a signature

To edit an already added signature, click/tap on in the signature's settings. This will open the canvas for you to continue writing your signature.

Rotating a signature

To rotate a signature, click/tap on or in the signature's settings to rotate it counter-clockwise or clockwise, repsectively.

Images

Adding an image allows you to draw your signature in another program/app and add it as an image to your application. It is recommended that the image has a transparent background.

Adding an image

To add an image, click/tap on in the top bar. Then select the image you wish to add using your system's file browser.

Scaling an image

To scale an image, edit the number next to in the image's settings.

Rotating an image

To rotate an image, click/tap on or in the image's settings to rotate it counter-clockwise or clockwise, repsectively.

Pages

Adding a new page

To add a new, blank page to your application (for additional space, if necessary), click/tap on in the top bar.

Removing a page

To remove the last added page, click/tap on in the top bar.

Note

This will not remove any elements that you have placed on the page.

Note

Any elements that are not on a page when the final PDF is downloaded will not be included in the document.

Downloading and uploading

Important

After sending us your PDF application, it is recommended to delete all saved .dat files as well as the PDF since they contain sensitive personal information. Be sure to delete them from your device's recycling bin or trash folder as well.

Download an in-progress application (can be edited)

To download an in-progress application so that you can complete it later (or on a different device), click/tap on in the top bar.

Note

You will be prompted to enter a password, which is highly recommended, as the file will contain your highly sensitive data (i.e. SSN) if you have entered it into your application. Be sure you remember your password, otherwise the progress on your application will be lost.

Important

Do not send us your application in this format; it will not be accepted. You must download your completed application as a PDF.

Upload an in-progress application (to continue editing)

To upload an in-progress application so you can continue working on it, click/tap on in the top bar. Select the .dat file you downloaded earlier, and enter your password to continue.

Download a complete application (as PDF)

To download your completed application to send to us, click/tap on in the top bar.

Important

The PDF format cannot be reuploaded and edited by this tool; you must download a .dat file if you want to save your application to work on later.

Note

Any elements that are not on a page when the final PDF is downloaded will not be included in the document. Be sure you check the PDF for errors before you send it to us!
